## Day 1 — [05/07/2026]

**Platform:** TryHackMe
**Room/Path:** Pre Security - Security Introduction

### What I covered
- Difference between Offensive Security and Defensive Security
- Offensive Security: thinking like an attacker to find weaknesses before real hackers do
- Defensive Security: protecting systems, detecting threats, and responding to attacks (SOC, monitoring, incident response)
- How both sides work together in a real security team

### Key takeaway
Offense finds the holes, defense closes and monitors them — understanding both sides makes you better at either one you specialize in later.
